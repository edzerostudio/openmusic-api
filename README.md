## About OpenMusic API

OpenMusic API is a Simple API for Albums & Songs management

## How To

Prepare your postgres database and configure `.env` file,  
then run this command:  
`npm install`  
`npm run migrate up`  
then use:  
`npm run start-dev` or `npm run start-prod`
